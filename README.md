# ios-image-trimmer
Set and trim square-and-rectangle-sized images

Using Xcode 11.2.1, Target iOS 13.2

---

*Features*

1) Set and save square/rectangle images

2) Camera (with pinch-to-zoom)

3) Trim (with pinch-to-zoom)
